# Projeto de Login e Cadastro - React + Vite 🚀

Este é um projeto de interface moderna para sistemas de autenticação, focado em design limpo e navegação fluida entre telas.

## 🛠️ Tecnologias Utilizadas

* **React**: Biblioteca para construção da interface.
* **Vite**: Ferramenta de build rápida para o desenvolvimento.
* **React Router Dom**: Gerenciamento de rotas e navegação.
* **CSS3**: Estilização personalizada com degradês e fontes modernas.

Siga os passos abaixo para configurar o ambiente em sua máquina local:
